# 點心粵語點字輸入法

> 點字書寫我心

點心粵語點字輸入法是一款專為視障人士度身訂造的粵語點字輸入解決方案，基於 Rime（中州韻）引擎開發。讓粵語使用者能夠透過點字，流暢地輸入中文，書寫內心所想。

## ✨ 核心功能

- **粵語點字輸入**：完整支援粵語盲文點字編碼
- **Rime 引擎驅動**：基於強大穩定的 Rime 引擎
- **一鍵安裝**：Windows 版本提供整合安裝包，開箱即用
- **無障礙優先設計**：由視障開發者親自開發與測試
- **跨平台潛力**：理論上可移植至 macOS、Android 等平台
- **開源免費**：完全開源，永久免費使用

## 📥 下載與安裝

### Windows 版本

點心粵語點字輸入法目前提供 Windows 一鍵安裝包。

**最新版本**：v1.1.0

**下載連結**：[Dimsum_Setup_v1.1.exe](https://github.com/dimsumgroup/dimsum-braille-ime/releases/download/v1.1.0/Dimsum_Setup_v1.1.exe)

### 安裝步驟

1. **下載安裝包**
   - 前往 [Release 頁面](https://github.com/dimsumgroup/dimsum-braille-ime/releases) 下載最新版本的 `Dimsum_Setup_v1.1.exe`

2. **執行安裝程式**
   - 雙擊下載的安裝檔案
   - 依照安裝精靈的指示完成安裝
   - 可能需要管理員權限

3. **切換至點心輸入法**
   - 在 Windows 工作列右下角的輸入法切換區選擇「點心粵語點字輸入法」
   - 首次使用時，Rime 引擎可能需要數秒鐘進行初始化

4. **開始輸入**
   - 輸入法啟動後，即可開始使用點字進行粵語輸入

## 📖 使用指南

### 盲文點位對應

點心輸入法採用標準盲文 6 點編碼系統。每個點字格由 6 個點位組成，對應鍵盤上的特定按鍵：

```
點1 → F    點2 → D    點3 → S
點4 → J    點5 → K    點6 → L
```

### 常見問題

**Q: 與普通粵拼輸入法有何不同？**
A: 點心輸入法使用盲文點字編碼，專為視障人士設計，無需依賴視覺反饋。

**Q: 是否支援廣東話語音？**
A: 目前版本不支援語音輸入，後續版本將考慮加入此功能。

**Q: 可以自訂詞庫嗎？**
A: 可以。基於 Rime 引擎，用戶可以自行調整詞庫與設定。

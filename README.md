# 點心粵語點字輸入法

> 點字書寫我心

點心粵語點字輸入法是一款專為香港視障朋友度身訂做、在 Windows 系統上完美運作的粵語點字打字工具。本輸入法採用先進的 Rime (中州韻) 引擎驅動，並完美支援 NVDA 螢幕閱讀器。

---

## 📥 快速開始

### 下載安裝

**最新版本**：v1.1.0

**下載連結**：[Dimsum_Setup_v1.1.exe](https://github.com/dimsumgroup/dimsum-braille-ime/releases/download/v1.1.0/Dimsum_Setup_v1.1.exe)

### 安裝步驟

1. 下載 `Dimsum_Setup_v1.1.exe` 檔案
2. 雙擊執行安裝程式
3. 按照安裝精靈的指示完成安裝
4. 安裝完成後，按下 **`Windows 鍵 + 空白鍵`** 切換輸入法
5. 當您聽到 NVDA 報讀「點心粵語點字」時，即可開始使用

詳細安裝說明請參考：[INSTALL.md](./INSTALL.md)

## 📖 使用指南

### 基本打字指法（並擊輸入）

本輸入法採用標準的「六點字」鍵盤排列：

```
左手： F (1點)、D (2點)、S (3點)
右手： J (4點)、K (5點)、L (6點)
```

**打字方式**：請「同時按下」對應點位的英文字母鍵，然後「同時放開」（此即為並擊）。

### 選字與上字方式

1. **直接上字**：按下「空白鍵 (Space)」即可直接送出第一個候選字
2. **瀏覽找字**：按「下方向鍵」或「H」鍵往下瀏覽；按「上方向鍵」或「G」鍵往上瀏覽
3. **快速選字**：按「1 到 9」的實體數字鍵，即可快速送出對應編號的候選字
4. **數字輸入**：直接使用鍵盤上方的實體數字鍵，並按下 Enter 鍵輸出

### 三大模式自由切換

按下 **`Control + `（反引號鍵）** 即可切換：

1. **點心粵語點字 (記憶模式)**【預設推薦】：具備智慧學習功能，常用字會自動排到前面
2. **點心粵語點字 (標準模式)**：關閉記憶功能，候選字順序永遠固定
3. **點心粵語點字 (一陽指模式)**：專為不習慣「六點並擊」的使用者設計

完整使用指南請參考：[USAGE_GUIDE.md](./USAGE_GUIDE.md)

## ✨ 核心特性

- **完美支援粵語點字編碼**：採用標準六點字系統
- **智慧學習功能**：自動記錄您的選字習慣與新詞
- **香港本地化**：收錄香港口語字與常用地名用字
- **NVDA 完美相容**：支援最新版 NVDA 螢幕閱讀器
- **開源免費**：完全開源，永久免費使用
- **長句連打**：強大的自動組句功能
- **客製化自由度高**：設定為純文字檔，輕鬆新增專屬字彙

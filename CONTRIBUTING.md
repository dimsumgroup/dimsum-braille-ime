# 貢獻指南

感謝您對點心粵語點字輸入法的興趣！我們歡迎各種形式的貢獻。

## 如何貢獻

### 報告 Bug

如果您發現了 Bug，請提交 Issue 並包含以下資訊：

- **清晰的標題**：簡明扼要地描述問題
- **詳細描述**：說明問題的具體情況
- **重現步驟**：列出重現 Bug 的步驟
- **預期行為**：說明應該發生什麼
- **實際行為**：說明實際發生了什麼
- **系統資訊**：Windows 版本、螢幕閱讀器等

### 提交功能建議

如果您有功能建議，請提交 Issue 並包含：

- **功能描述**：清楚地描述您想要的功能
- **使用場景**：解釋這個功能為什麼有用
- **可能的實現方式**：如果您有想法的話

### 貢獻代碼

1. **Fork 本倉庫**
2. **建立功能分支**：`git checkout -b feature/your-feature`
3. **提交更改**：`git commit -am 'Add your feature'`
4. **推送到分支**：`git push origin feature/your-feature`
5. **提交 Pull Request**

## 代碼風格

請遵循以下代碼風格：

- 使用 UTF-8 編碼
- 提交前進行測試
- 編寫清晰的提交訊息
